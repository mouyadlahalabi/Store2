

 مشروع Redmi - تعليمات تشغيل المشروع:

## طريقة التشغيل
1. نسخ المشروع:
   git clone https://github.com/mouyadlahalabi/Store2.git
2. إنشاء بيئة افتراضية:
   python -m venv venv
3. تفعيل البيئة:
   venv\Scripts\activate  # Windows
   source venv/bin/activate  # Linux/macOS
4. تثبيت المتطلبات:
   pip install -r requirements.txt
5. تشغيل المشروع:
   python manage.py runserver
